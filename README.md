# BAO (Boite A Outils)
Boite A Outils est un logiciel d'aide au dépannage informatique pour les professionnels

## Compatibilité
* Windows 8.1, Windows 10
* Compatibilité partielle avec Windows Vista et Windows 7

## Execution
Ouvrez run.bat
